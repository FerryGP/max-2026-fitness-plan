# Max成长 · 项目归档与暑期体测计划

当前包含 Max 暑期体测计划网页工具。打开 `01_体测训练/index.html` 即可使用。

这个文件夹后续作为 Max 相关项目的归档入口，新的 Max 成长、学习、训练、活动项目可以继续放在这里。

功能：

- 每日训练计划：每天 3-4 件事，以减重和跳绳为主线；跳绳目标 180+ 留余量，仰卧起坐按 49+ 满分线练，每天拉伸坐位体前屈方向。
- 每日打卡：保存完成项和备注。
- 成绩记录：记录肺活量、50 米、坐位体前屈、跳绳、仰卧起坐、身高、体重，并自动计算 BMI。
- 每天提醒：页面打开时可开启浏览器通知；也可以生成 `暑期体测训练提醒.ics` 导入系统日历。

默认暑期周期为 2026-07-06 起 8 周，可在页面顶部修改开始日期和提醒时间。

个人身高、体重和体测成绩不预置在公开网页中。首次打开后可手动录入，记录仅保存在当前设备的浏览器中。

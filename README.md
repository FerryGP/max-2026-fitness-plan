# Max · 二六年暑期体测满分计划

打开 `index.html` 即可使用。

功能：

- 每日训练计划：每天 3-4 件事，以减重和跳绳为主线，顺带提升 50 米、仰卧起坐、心肺和协调。
- 每日打卡：保存完成项和备注。
- 成绩记录：记录肺活量、50 米、坐位体前屈、跳绳、仰卧起坐、身高、体重，并自动计算 BMI。
- 每天提醒：页面打开时可开启浏览器通知；也可以生成 `暑期体测训练提醒.ics` 导入系统日历。

默认暑期周期为 2026-07-06 起 8 周，可在页面顶部修改开始日期和提醒时间。

当前基线：2026-07-09 测试，身高 140cm、体重按 45kg 估算，BMI 约 23.0；肺活量 2680，50 米 11.10 秒按 60 分计，坐位体前屈 18cm，跳绳最好 167 个，仰卧起坐 42 个，估分约 100 分，中考四年级过程性分 10/10。

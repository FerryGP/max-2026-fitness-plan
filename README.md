# Max · 二六年暑期体测满分计划

打开 `index.html` 即可使用。

功能：

- 每日训练计划：围绕 50 米提速、BMI 管理、跳绳保持和核心力量。
- 每日打卡：保存完成项和备注。
- 成绩记录：记录 50 米、跳绳、仰卧起坐、身高、体重，并自动计算 BMI。
- 每天提醒：页面打开时可开启浏览器通知；也可以生成 `暑期体测训练提醒.ics` 导入系统日历。

默认暑期周期为 2026-07-06 起 8 周，可在页面顶部修改开始日期和提醒时间。

当前基线：身高 140cm、体重 45kg，BMI 约 23.0；仰卧起坐 39 个，跳绳发挥好可到 160 个，50 米沿用旧记录 10.10 秒作为暑期起点。

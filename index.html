一、每天的大原则

早餐

必须吃。以鸡蛋、牛奶/豆浆、粗粮主食和水果为主，帮助白天不乱吃。

午餐

正常吃，但主食换成玉米、红薯、燕麦、杂粮饭、全麦馒头，量控制在七分饱左右。

晚餐

清淡一点，海鲜优先清蒸、白灼、煮汤，主食少量，尽量不吃油炸和重油做法。

加餐

如果饿，优先水果、无糖酸奶、牛奶。不要拿饼干、薯片、甜饮料顶饿。

  • 主食不吃白米饭，改成粗粮、杂粮、玉米、红薯、燕麦、全麦面包。
  • 每餐都要有蛋白质，比如蒸鸡蛋、鱼片、虾、瘦牛肉、豆腐。
  • 每天喝 300–500ml 纯牛奶或等量奶制品；两盒合计不超过 500ml 时可以，不能用牛奶代替正餐。
  • 海鲜可以吃，但尽量清蒸、白灼、清炖，少油少酱。
  • 饮料只喝水、牛奶、无糖豆浆，不喝奶茶和甜饮料。
  • 晚饭别吃太晚，睡前不加夜宵。

Max 自己判断的 5 条标准

吃饭

每餐一掌心蛋白质、两拳蔬菜、一拳粗粮，吃到不饿、不撑。

饮品

每天牛奶 300–500ml，其余主要喝水，不喝甜饮料。

训练

每天累计活动约 60 分钟；能说短句、不能轻松唱歌，强度就够了。

停止

头晕、恶心、胸闷、膝踝疼或动作变形，马上停下来告诉家长。

复盘

每周固定一天晨起称重；精神、睡眠和体测成绩正常才继续。

记住：不饿肚子、不喝甜饮料、每天动够、身体不难受、成绩不下降。

二、按六周推进

第 1 周

先把白米饭换掉，建立粗粮主食习惯，控制零食和饮料。

第 2 周

午餐和晚餐都稳住，晚餐主食再少一点，海鲜尽量清蒸白灼。

第 3 周

继续减油减糖，训练日多一点蛋白质，恢复日减少主食总量。

第 4 周

保持三餐稳定，结合周测结果调整份量，不通过挨饿追赶数字。

第 5 周

继续清淡饮食和规律活动，重点观察跳绳、跑步成绩是否保持。

第 6 周

进入体测准备期，不再激进减量，保证睡眠、饮水和训练恢复。

三、一周 7 天示例

周一

早：蒸鸡蛋 + 牛奶 + 燕麦；中：杂粮饭 + 番茄牛肉 + 西兰花；晚：番茄鱼片汤 + 红薯。

周二

早：全麦面包 + 蒸鸡蛋 + 牛奶；中：玉米 + 白灼虾 + 西兰花；晚：番茄豆腐汤 + 小半根玉米。

周三

早：牛奶 + 蒸鸡蛋 + 小杂粮馒头；中：杂粮饭 + 番茄鱼片 + 青菜；晚：西兰花牛肉 + 黄瓜。

周四

早：牛奶 + 燕麦 + 香蕉；中：红薯 + 番茄牛肉 + 西兰花；晚：番茄蒸蛋 + 少量全麦面包。

周五

早：蒸鸡蛋 + 无糖酸奶 + 玉米；中：杂粮饭 + 番茄鱼片 + 西兰花；晚：虾仁蒸蛋 + 生菜。

周六

早：牛奶 + 蒸鸡蛋 + 红薯；中：粗粮饭 + 西兰花牛肉 + 番茄；晚：番茄豆腐鱼片汤。

周日

早：燕麦粥 + 蒸鸡蛋 + 牛奶；中：玉米 + 番茄牛肉 + 西兰花；晚：番茄鱼片汤 + 少量粗粮。
